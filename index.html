Annotations

Flèche, rectangle, cercle ou texte. Les annotations peuvent être déplacées, copiées, supprimées et ajustées.

Commandes

Souris
- Molette : zoom
- Molette maintenue : déplacement de la vue
- Clic-glisser sur sélection : déplacer
- Ctrl/Maj + clic : ajouter/retirer
- Clic-glisser vide : fenêtre de sélection

Clavier
- Ctrl+Z / Ctrl+Y : annuler/rétablir
- Ctrl+C / Ctrl+V : copier/coller
- R : rotation selon l'angle choisi
- M : miroir
- F2 : modifier texte sélectionné
- Suppr : supprimer sélection
- Échap : annuler mode actif
